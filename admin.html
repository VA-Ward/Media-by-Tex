How do I get an access token? (one-time, 2 minutes)
  1. Go to github.com/settings/personal-access-tokens/new
  2. Name it anything (e.g. "Tex gallery"). Set an expiry (or "No expiration").
  3. Under Repository access, choose Only select repositories and pick your website repo.
  4. Under Permissions → Repository permissions → Contents, set it to Read and write.
  5. Click Generate token, copy it, and paste it above. Keep it private — you can revoke it anytime.

Your token is kept only in this browser tab and is never saved to your site.
